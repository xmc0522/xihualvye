### 修复：启动后白屏问题
- **原因**：循环依赖 → `menu.ts` 同步 import `Dashboard.vue` → `Dashboard.ts` import `menuArr` from `menu.ts`，导致 `menuArr` 在尚未初始化时被读取，返回 undefined
- **修复**：将 `menu.ts` 中 Dashboard 改为懒加载 `const Dashboard = () => import('@/views/首页/Dashboard.vue')`，打断循环依赖链
